# Создание пакетов

В этом разделе описано, как создавать пакеты для APKG.

## Структура пакета

Пакет APKG состоит из:

1. `package.json` - Метаданные пакета
2. `files/` - Файлы пакета
3. `scripts/` - Скрипты установки
   - `pre-install.sh`
   - `post-install.sh`
   - `pre-deinstall.sh`
   - `post-deinstall.sh`
