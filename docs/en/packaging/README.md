# Creating Packages

This section describes how to create packages for APKG.

## Package Structure

An APKG package consists of:

1. `package.json` - Package metadata
2. `files/` - Package files
3. `scripts/` - Installation scripts
   - `pre-install.sh`
   - `post-install.sh`
   - `pre-deinstall.sh`
   - `post-deinstall.sh`
